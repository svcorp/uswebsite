Measurement & Analysis Software

Sound Level Meter (SLM)

  • Standards : IEC61672-1 Class1, IEC60651 Type1
  • Frequency Range : 0.5 ~ 25kHz
  • Freq. Weighting : A, B, C, Z
  • Time Weighting : Fast, Slow, Impulse
  • Integration Time : 0.125, 0.25, 0.5, 1, 2, 5, 10 sec
  • Dynamic Range, Linearity : 120dB more
  • Display Mode : Digital Value, Compare, Graph
  • Detector : Selectable Synchronous 2 Detectors
  • Measurement Value : Sound Pressure Level(Lp), Equivalent Noise Level(Leq),
    Noise Exposure Level(Le), Max, Min SPL, Statistical Noise Level(Ln), Peak Noise Level(Lpeak)

Frequency Analysis (FFT)

  • Frequency : Selectable 200Hz ~ 25kHz
  • Buffer Size : Selectable 400, 800, 1600, 3200, 6400 (Max. 3000 Lines)
  • Window : Rectangular, Hanning, Flattop
  • Average : Linear, Exponential, Peak
  • Weight : A, B, C
  • Auto Peak Detection Function & Y-axis auto scaling
  • Harmonic Cursor
  • Post Process : FFT Analysis after Variables Modification such as Band ,
    Buffer Size, Window, Averaging, Weight Compensation

FFT-based Octave Analysis (OCT)

  • Frequency : 25kHz
  • Buffer Size : 6400
  • Window : Rectangular, Hanning, Flattop
  • Average : Linear, Exponential, Peak
  • Weight : A, B, C
  • Y-axis Auto Scaling
  • Post Process : FFT Analysis after Variables Modification
    such as Band , Buffer Size, Window, Averaging, Weight Compensation
